# Calender Manager Agent

## Overview

This project is a calendar manager agent that uses LLMs to help users manage their calender events using natural language.

## Features

- Add a new event to the calender
- Delete an event from the calender
- Get all events from the calender
- Agent suggests events re-ordering if needed.


## Tech Stack

**Backend**
- API: FastAPI
- Database: PostgreSQL
- Cache: Redis
- Vector Database: PgVector
- LLM: OpenAI
- Tracing & Monitoring: Langfuse

**Frontend**
- NextJS
- TailwindCSS
- Shadcn UI
